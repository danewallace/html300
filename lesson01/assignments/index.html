Using jQuery and ES6+ Methods

forEach

Calls a function for each element in the array.

This function will output keys and values from a pre-created array saved in /scripts.js

Filter

Reduce the set of matched elements to those that match the selector or pass the function's test.

Submit a number. The output will display all numbers in the original array that are greater than the submitted number.

Map

Pass each element in the current matched set through a function, producing a new jQuery object containing the return values.

Submit a number. The output will display the sum of the submitted number and the array's original number.

Includes

Determines whether a string contains the characters of a specified string.

Submit a string. If the submitted string is included within the sample text, the output will be true. Otherwise, it will be false.

Sample text: Who's the more foolish, the fool, or the fool who follows him?
